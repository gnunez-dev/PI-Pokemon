Uso
===========

Para ejecutarlo hay que crear una base de datos en postgres con el nombre de

```
pokemon
```

Luego hacer 

```
npm install
npm start
```

hay que hacerlo en los directorios api y client.

El back carga en el puerto 3001
```
localhost:3001
```

El front carga en el 3000

```
localhost:3000
```
